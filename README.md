# Project-2-semester
Проект: Список контактов
Основная задача понять как построить взаимодействие между двумя классами. Пока что одним классом будет весь список контактов, а другой класс отвечает за отдельный контакт, тоесть он будет вызывать определенный контакт, создавать его и соответсвенно удалять. Для идей с формами думал добавить основную форму, через которую можно будет перейти в другие формы, к примеру список контактов и создание нового контакта. НУ пока все.
