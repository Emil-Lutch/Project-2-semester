# Contacts
## Что может приложение:
  ▪ Пользователь вводит имя, фамилию и номер телефона.  
  ▪ Может просмотреть весь свой список контактов и редактировать его.
## Принцип работы:
  ▪ Пользователь вводит в соответствующие поля соответствующую информацию об контакте и нажатием кнопки сохраняет его.  
  ▪ Процесс создания контакта происходит на отдельной форме.  
  ▪ Добавленный или уже существующий контакт отображается в listBox и он хранится в txt файле.  
  ▪ Просмотр и редактирование контактов происходит также в отдельной форме.
## Руководство администратора
  ▪ Visual Studio 2022  
  ▪ Открыть решение ```Contacts.sln```  
  ▪ Запустить проект  
## Руководство пользователя
  Запустить программу  
  
  1.В поле создания контакта:  
    ▪ Внести данные  
    ▪ Нажать кнопку "Добавить"  
    
  2.Список контактов:  
    ▪ Если надо, нажать "Удалить"  
    ▪ Возможность вернуться в прошлую форму через кнопку "Назад"
## Структура классов
```Contact``` - хранит данные контакта (имя, фамилия, номер)  
```CheckInfo``` - управление списком контактов (добавить, удалить)  
```ContactForm``` - форма создания контакта   
```ContactListForm``` - форма удаления и списка контактов
## Работа с данными
Данные сохраняются в файл: ```contacts.txt```  
Формат хранения:  
▪ Имя; Фамилия; Номер  
При запуске данные автоматически загружаются из файла
