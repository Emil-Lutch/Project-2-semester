# Project-2-semester
Проект: Список контактов
Основная задача понять как построить взаимодействие между двумя классами. Пока что одним классом будет весь список имен контактов, а другой класс отвечает за их номера. Классы будут связанны между собой с помощью двух переменных: имя контакта и номер контакта. Также будет документ хранящий контакты, откуда можно будет вызвать определенный контакт и узнать его номер телефона. Для идей с формами думал добавить основную форму, через которую можно будет перейти в другие формы. Создать отдельную форму для просмотра всего списка контактов, также с добавлением нового контакта в архив.
